## 特点
编译型语言（一次运行）
```bash
go run helloworld.go
```
保存编译的程序和结果
```bash
go build helloworld.go
```
换行符就是分号  
格式化的工具
```bash
gofmt helloworld.go 
go fmt helloworld.go
```
